Bem vindo!

Este é um sistema de cadastramento de formulários automatizado. Com ele você poderá configurar de forma fácil um novo formulário para sua aplicação.

Utiliza o painel de navegação ao lado para realizar iniciar o cadastramento.

Como utilizar?

A ferramenta segue uma hierarquia de cadastramento, ou seja os elementos do questionário devem ser cadastrados na seguinte ordem:

  1. Questionário;
  2. Seção;
  3. Perguntas nível 1;
  4. Perguntas nível 2;
  5. Perguntas nível 3.

As perguntas com múltiplas opções de resposta (exceto aquelas dos tipos "Sim/Não" e "Sim/Não/Não se aplica") devem seguir a seguinte ordem para cadastramento:

  1. Cadastrar opções;
  2. Vincular as opções nas perguntas.

Ordem

Os elementos "Seção" e "Perguntas" possuem um atributo chamado "Ordem" que determina a posição que o elemento aparece na interface do usuário. Este atributo é representado por um número inteiro gerado automaticamente durante a criação do elemento, mas que pode ser alterado a qualquer momento.

Exibição

Todos os elementos possuem o atributo "Ativo" disponível para configuração nas páginas de edição. Os elementos recém criados são marcados como inativos e só podem ser ativados após atenderem às condições mínimas exigidas.
